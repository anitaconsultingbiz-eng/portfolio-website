Learning & Build Journal

Learning frontend development through real projects.

This page documents my hands-on learning process as I build real projects using HTML, CSS, GitHub, and frontend development tools.

Project 01

Portfolio Website

Built and published a personal portfolio website using HTML, CSS, VS Code, GitHub, and GitHub Pages.

What I Practiced

  • HTML page structure
  • CSS Grid and Flexbox
  • Responsive design
  • Navigation links
  • GitHub Pages deployment

Challenges I Worked Through

  • Fixing card layouts that stretched too wide
  • Improving spacing between sections
  • Connecting buttons to page sections and email links
  • Publishing updates through GitHub

Lesson Learned

I learned that frontend development is not only about writing code. Small decisions about spacing, layout, structure, and hierarchy can make a website feel more professional and easier to use.
